# Meu Site

Um site estático moderno criado com HTML, CSS e JavaScript.

## Estrutura do Projeto

```
site/
├── index.html      # Página principal
├── styles.css      # Estilos do site
├── script.js       # Lógica JavaScript
└── README.md       # Este arquivo
```

## Características

- ✨ Design responsivo (mobile-friendly)
- 🎨 Estilo moderno com gradientes
- ⚡ Navegação suave (smooth scrolling)
- 📱 Layout adaptativo para diferentes telas
- 📧 Formulário de contato funcional

## Como Usar

1. Abra o arquivo `index.html` no navegador
2. Ou use uma extensão VS Code como "Live Server" para visualizar em tempo real

### Com Live Server

- Instale a extensão "Live Server" no VS Code
- Clique com botão direito em `index.html` e selecione "Open with Live Server"
- O site abrirá em `http://localhost:5500`

## Personalização

### Trocar cores

Edite `styles.css` e procure por cores como `#667eea`, `#764ba2` ou `#007bff`

### Alterar logo e título

Edite `index.html` e procure por "Meu Site"

### Adicionar novas seções

Copie uma seção existente em `index.html` e customize conforme necessário

## Recursos

- HTML5 semântico
- CSS Grid e Flexbox
- JavaScript vanilla (sem dependências)
